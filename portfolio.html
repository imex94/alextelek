Tili-Toli

A strategic puzzle game called Tili-Toli. There are buttons in 3x3, 4x4, or 5x5 squares with one empty space each. The application shuffle the buttons, using heuristic search to find the best solution, and the user has to find the fastest way of solving the puzzle.
